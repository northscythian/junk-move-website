# Чек-лист для запуска бизнеса

## 1. Регистрация
- [ ] Выбрать название бизнеса
- [ ] Проверить название в NUANS
- [ ] Зарегистрировать в провинции
- [ ] Получить Business Number (BN)
- [ ] Зарегистрироваться в CRA для GST/HST

## 2. Страховка
- [ ] Commercial General Liability
- [ ] Commercial Auto Insurance
- [ ] Workers' Compensation (WSIB / WCB)

## 3. Оборудование
- [ ] Купить грузовик / трейлер
- [ ] Купить инструменты (ремни, тележки, лебёдки)
- [ ] Закупить расходники (коробки, скотч, плёнка)

## 4. Маркетинг
- [ ] Создать сайт / лендинг
- [ ] Зарегистрировать Google Business Profile
- [ ] Настроить соцсети (Instagram, TikTok, Facebook)
- [ ] Напечатать визитки и флаеры

## 5. Финансы
- [ ] Открыть бизнес-счёт
- [ ] Настроить бухгалтерию (Wave, QuickBooks)
- [ ] Установить цены и формы оплаты
